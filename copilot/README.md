# GitHub Copilot 文档

## 简介
GitHub Copilot 是微软和 github 联合开发的 AI 编程工具，目前主要集成在 VSCode 和 VS 中, JetBrains 系列 IDE 只有有限的支持

## 功能特点
- 🤖 AI 驱动的代码建议和补全
- 💡 上下文感知的代码生成
- 🔍 多语言支持
- 🛠️ 与主流 IDE 集成
- 🌐 实时代码分析
- 🔄 持续学习和改进

## 价格
- 个人版：$10/月 或 $100/年
