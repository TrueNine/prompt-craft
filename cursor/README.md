# Cursor

Cursor 是一个 AI 驱动的代码编辑器，总部位于美国。它集成了 AI 技术，帮助开发者编写、理解和调试代码。

团队订阅计划：$20/月/用户

## 主要特点

- 🤖 内置 AI 助手，支持自然语言交互
- 💡 智能代码补全和建议
- 🔍 强大的代码搜索和导航功能
- 🛠️ 内置代码重构和优化建议
- 🌐 支持多种编程语言和框架
- 🔄 实时代码分析和错误检测
